# SmartFix

SmartFix is a native iOS application that connects users with technicians for household repair services.

The application allows users to report appliance faults, upload images of the problem, and receive offers from technicians. Technicians can review service requests and submit repair offers based on the reported issue.

## Features

- User and Technician authentication
- Visual fault reporting with photo upload
- Competitive bidding system for repair offers
- Technician subscription system
- In-app messaging between users and technicians

## Technologies

- Swift
- UIKit
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- MVVM Architecture

## Architecture

The application follows the **MVVM (Model–View–ViewModel)** architecture to separate UI, business logic, and data layers. This structure improves maintainability and scalability of the project.
